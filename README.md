Log Analyzer

This is a simple Python log analyzer created to practice programming and basic cybersecurity skills.

The tool can:
- Count log levels INFO WARNING and ERROR
- Show lines with specific log levels
- Extract IP addresses from logs
- Count how many times each IP appears
- Show the most frequent IP address

Features
- Input custom log file path
- Log statistics summary
- IP address analysis
- Error handling for incorrect file paths
- Simple command line menu

Technologies
Python
re module
file handling

How to run

python log_analyzer.py

Example menu

1 - Show log statistics
2 - Show ERROR lines
3 - Show INFO lines
4 - Show WARNING lines
5 - Show IP statistics
6 - Show most frequent IP
7 - Exit
